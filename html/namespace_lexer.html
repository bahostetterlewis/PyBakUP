Detailed Description

This is the parser that is used to make custom backup conditions.

This package will use ply(Python Lex and Yacc) to create a method for parsing the user generated backup conditions (hopefully making it easy to extend as desired. It will take a string, parse it, and return a boolean (true or false) which will allow for the program to decide if a backup is necessary

Note:
Tokens have the following attributes: type, value, lexpos
